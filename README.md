# made-with-loops
This repository contains the source code and learning journal for building the **made-with-loops** website.

The project is focused on creating a real-world website while learning core concepts of frontend development, backend development, and basic DevOps practices over time.

## Project Goals
- Build a functional website for the made-with-loops brand
- Learn and apply frontend development concepts
- Gradually introduce backend functionality
- Document learning progress through a daily journal
- Understand basic DevOps and deployment workflows

## Tech Stack (Evolving)
- Frontend: HTML, CSS, JavaScript
- Backend: To be added
- Database: To be added
- DevOps: To be added
